# 视觉风格定义 — GitHub Trending 2026-05-13

## 情绪提炼

| 维度 | 描述 |
|------|------|
| 主题 | GitHub 今日最热门开源项目盘点 |
| 情绪基调 | 兴奋发现、科技前沿、朋友分享好东西 |
| 情绪弧线 | 震撼数据钩子 → 好奇探索各项目 → 行动号召 |
| 节奏感 | 紧凑，每个项目 3-4 秒 |
| 文化调性 | 现代科技 |

## 视觉风格

| 属性 | 值 |
|------|---|
| 风格方向 | 科技未来主义 — 深色主题 + 霓虹线条 |
| 主背景 | 深色渐变 #0d1117 → #161b22（GitHub Dark 主题色） |
| 强调色 | #00e5a0（霓虹绿）+ #58a6ff（GitHub 蓝） |
| 文字色 | #ffffff（主标题）+ #8b949e（副文字） |
| 数据高亮 | #f0883e（橙色用于 Star 数字） |
| 字体 | 无衬线体，标题用粗体 |
| 布局 | 竖屏 1080×1920，内容居中，底部留安全区 |

## 配乐方向

| 属性 | 值 |
|------|---|
| 风格 | 电子科技 / Lo-fi beats |
| 情绪 | 紧凑、有推进感、不过于激烈 |
| 关键词 | "tech electronic upbeat no copyright", "lo-fi future beats" |
| 音量 | BGM 低于旁白，作为氛围铺垫 |

## 素材需求预判

| 需求 | 方案 |
|------|------|
| 项目名称展示 | 纯 CSS 文字动画 + 霓虹发光效果 |
| Star 数据展示 | CSS 数字动画 |
| 语言标签 | CSS 圆角标签 + 对应颜色 |
| 背景氛围 | CSS 渐变 + 粒子/网格线效果 |
| 项目图标 | 纯 CSS 图标（代码符号、齿轮等） |

> 不需要外部图片素材，纯 CSS + 动画即可实现所有视觉效果。
