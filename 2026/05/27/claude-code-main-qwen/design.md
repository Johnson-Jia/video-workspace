# Design: Claude Code 深度解析

## 项目信息
- **主题**: Claude Code — 从源码泄露看 AI 编码工具的终极形态
- **模式**: 单项目深度解析（8 场景）
- **目标时长**: 50-55 秒
- **内容来源**: 源码泄露分析报告 + 使用指南 + 官方文档

## 风格方向

### 导演必答
| Q | 回答 |
|---|------|
| Q1 情感内核 | **震撼** → **好奇** → **掌控感** |
| Q2 情感节拍 | grab(震撼数据) → build(架构揭秘) → reveal(隐藏功能) → climax(护城河洞察) → settle(实用指南) → summon(行动) |
| Q3 视觉手段 | 暗色科技基底 + 源码绿/Anthropic 橙双强调色 + 终端风格等宽字体 |
| Q4 视觉反差 | hook 高密度数据冲击 → what 大留白定义 → how 架构图解 → 隐藏功能逐条揭示 |
| Q5 视觉焦点 | 每个场景一个核心数字或关键词，100px+ 冲击层 |

### 配色方案
- **背景**: 深蓝黑渐变 `#0A0E1A` → `#111827`
- **主文字**: 白色 `#FFFFFF` / 浅灰 `#9CA3AF`
- **强调色 A（数据/钩子）**: Anthropic 橙 `#D97757`
- **强调色 B（技术/代码）**: 终端绿 `#34D399`
- **强调色 C（隐藏/神秘）**: 紫色 `#8B5CF6`
- **光晕**: 暖橙左上 + 冷青右下（标准科技风）
- **隐藏功能场景**: 切换为紫色暖光 + 深蓝冷光（神秘感）

### 字体
- 标题：无衬线粗体（系统默认）
- 代码/数字：等宽字体（ JetBrains Mono 风格）

### 情绪曲线
```
grab ─── build ────── reveal ──── climax ─── settle ── summon
 震撼     好奇拆解    惊喜揭秘    行业洞察    实用落地   行动号召
  1.0      0.6        0.7         0.9        0.5       0.4
```

### 幽默策略
- 技术类视频，幽默偏轻
- 在 build/settle 节拍使用类比和冷知识梗
- "51 万行代码，比大多数公司的整个代码库都大"
- "终端里的拓麻歌子——写代码养宠物"
- "禁止甩锅式委派——Claude 比你的经理更懂管理"

### 沉浸模式
`hyper-pace`（AI / Agent 主题）— 快速剪辑 + 密集粒子 + 霓虹青

## Storyboard（8 场景）

| # | 场景 | 类型 | 时长 | 节拍 | 旁白摘要 |
|---|------|------|------|------|---------|
| 1 | hook | hook | 4s | grab | 51 万行源码泄露，暴露了 Anthropic 最大的秘密 |
| 2 | what | what | 6s | build | 它不是代码补全，是一个有 53 个工具的自主编程代理 |
| 3 | how | how | 7s | build | 三层门控体系：编译开关、用户类型、远程配置 |
| 4 | hidden | capabilities | 10s | reveal | 七大隐藏功能：从电子宠物到永不关机的 AI 助手 |
| 5 | architecture | tech | 8s | climax | 护城河不是模型，是 51 万行编排层代码 |
| 6 | usecases | usecases | 8s | settle | 终端、IDE、手机、CI/CD，全平台覆盖 |
| 7 | config | privacy | 6s | settle | 权限白名单、MCP 扩展、中转 API，企业级安全 |
| 8 | cta | cta | 4s | summon | 搜索 Claude Code，开启 AI 编程新范式 |
