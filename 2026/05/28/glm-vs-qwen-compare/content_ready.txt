# GLM-5.1 vs Qwen3.7-max 大模型实战对比

## 背景

公司从三月初全面施行「研发全员AI提效战略」，上百号产研小伙伴全部接入 GLM-5.1。
整体使用下来提效明显，但前期大家很辛苦——对新事物的接受度问题，大部分人本能是抗拒的，
而且 AI 产出的代码让程序员去审查、去 review 超级累人，综合下来就是学习所付出的成本。
不过整体下来，AI 战略提效项目推进非常健康，现在全研发人员都已经完全能驾驭 AI。

## 切换契机

Qwen3.7-max 发布，阿里云销售强力推荐，并给公司申请了 80 亿免费 tokens 用量。
在这种情况下大家就全部换成阿里云 Qwen3.7-max 开始验证大模型质量。
先说结论：Qwen3.7-max 在编码层面不如 GLM-5.1。

## 实测对比案例

### 案例 1：Claude Code 接入 Superpowers 安装

任务：让 Claude Code 自己安装 Superpowers 技能包。

- **GLM-5.1**：先查找 Claude guide 文档，然后搜索找到安装方法，给出正确安装指令。
- **Qwen3.7-max**：跑了半天，直接给出一个不知道 Superpowers 是什么的回答。

### 案例 2：视频生成自动化流水线 4 组对照实验

同一视频主题（Claude Code 源码泄露深度解析），两个分支架构（main / qwen-upgrade）x 两个大模型（GLM-5.1 / Qwen3.7），共 4 组对照。

#### 基本数据

| 指标 | A: qwen-upgrade+Qwen | B: qwen-upgrade+GLM | C: main+GLM | D: main+Qwen |
|------|---------------------|---------------------|-------------|-------------|
| 视频时长 | 75s（偏长） | 47s（偏短） | 55s（理想） | 56s（理想） |
| 文件大小 | 5.6 MB | 9.0 MB | 27 MB | 34 MB |
| HTML 行数 | 617 | 325 | 457 | 533 |
| 特效类型 | 0（仅静态圆点） | 0 | 6+ | 5+ |

#### 综合评分（满分 35 分）

| 维度 | A (qwen-upgrade+Qwen) | B (qwen-upgrade+GLM) | C (main+GLM) | D (main+Qwen) |
|------|----------------------|---------------------|-------------|-------------|
| 设计文档 | 2.0 | 3.5 | **5.0** | 3.0 |
| 旁白文案 | **4.5** | 3.0 | 4.0 | 3.5 |
| HTML 视觉 | 3.0 | 2.0 | **5.0** | 4.0 |
| FX/动画 | 2.0 | 1.5 | **5.0** | 4.0 |
| 抖音文案 | **4.5** | **4.5** | 3.5 | 2.0 |
| 时长合理性 | 2.0 | 3.0 | **5.0** | **5.0** |
| **总分** | **21.5** | **21.0** | **32.0** | **24.0** |

#### 关键发现

1. **模型差异 > 架构差异**：同一分支下换模型的视觉差异达 2-3 倍，同一模型换分支的文案差异仅 10-20%。
2. **GLM-5.1 是视觉引擎，Qwen3.7 是文案引擎**：GLM 擅长 HTML 工程和动画系统，Qwen 擅长文案节奏和传播策略。
3. **main 分支的指令框架更成熟**：导演必答、情绪曲线注解、幽默策略等框架有效提升了 GLM 的产出上限。
4. **理想组合是拆不开的**：GLM 做视觉 + Qwen 做文案能产生最优结果，但当前管线是一个模型从头到尾执行。

#### GLM-5.1 特点

擅长：
- HTML 视觉工程碾压级领先：角色系统、phase 切换、seeded 粒子
- 元数据结构化最完整：visual_phases、contrarian_angle、character_expression
- 设计文档深度最好：导演必答、精确配色规则
- FX 动画类型最丰富（6+ 种 CSS 动画）

弱点：
- 旁白偶有遗漏
- 抖音文案版本少、传播力设计不足

#### Qwen3.7 特点

擅长：
- 旁白节奏最紧凑，hook 最抓人
- 信息密度最高：更少字传达更多数据
- 抖音文案多版本 + 合规自查

弱点：
- HTML 视觉偏简单，特效层弱
- 无角色覆盖层、无语音气泡
- 设计文档结构偏薄

## 核心结论

不要被营销话术迷惑，实际编码场景下 GLM-5.1 明显优于 Qwen3.7-max。
智谱早就进入国产大模型第一梯队，Qwen3.7-max 号称国产第一进全球前三，实测并非如此。
选择大模型要看实战场景，而不是看营销排名。
